Bienvenido Gracias por elegir Express Bus Tickets


Introducción

Express Bus Tickets es una herramienta para la gestión de buses y venta de boletos, aplicado a una empresa de transporte.

¿Qué permite Express Bus Tickets?

- Express Bus Tickets facilita y mejora la venta de boletos en una empresa de transporte.

- Express Bus Tickets Permite tener un mayor control sobre las actividades que se desarrollan los usuarios.

- Express Bus Tickets permite trabajar en tiempo real (On-Line), utilizando la red de internet.

- Express Bus Tickets permite la comunicación instantanea entre terminales (sucursales).

- Express Bus Tickets esta optimizado para no consumir muchos recursos.

- Express Bus Tickets permite realizar copias de seguridad de la Base de Datos.

¿Qué Necesito para utilizar Express Bus Ticktes?

- Se requiere un web hosting o servidor web, con soporte (php5, mysql5).

- Una computadora con acceso a internet.

- Una impresora para la impresión del boleto.

- Se puede probar en un servidor local, pero no se podra aprobechar todo su potencial.

Caracteristicas de Express Bus Tickets

Express Bus Tickets es un software libre de código abierto, basando en la versión anteriror Express Bus, esta creado con la plataforma de programación utilizando el lenguaje php, usando como gestor de base de datos Mysql, asi mismo utiliza una interfaz gráfica basado en Joomla v1.6, este software aun se encuentra en face Beta, por lo que el creador de este programa no se responsabiliza por los inconvenientes que se pueden producir.

De la Interfaz Gráfica

Express Bus Tickets Utiliza una interfaz gráfica muy amigable, esta interfaz esta basado en la interfaz de administración de Joomla versión 1.6.

Límites del Programa

- Express Bus Tickets no contempla el área contable.

- Express Bus Tickets no administra el personal de trabajo, como ser: sueldos, asistencia, licencias, cambios, etc.

- Express Bus Tickets no contempla al personal de viaje: conductor, relevo, asafata, técnicos.

¿Qué ventajas ofrece Express Bus Tickets?

- Mejora la atensión al cliente rápido acceso a la información.

- Mejor control de errores, ventas y reservas.

- Evita la duplicidad de asientos vendidos.

- Permite tener un control sobre los usuarios del sistema.

- Permite crear un reporte económico diario de los servicios.

- Evita tener que manejar libros de registro o exesiva papelería.

- Permite trabajar en línea en tiempo real manteniendo una comunicación entre terminales.

- Optimiza el trabajo y mejara el rendimiento de tiempo.
